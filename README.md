# reproducing-of-RGCL
本仓库包含了论文《Improving Hateful Meme Detection through Retrieval-Guided Contrastive Learning, ACL 2024.》的代码、数据、测试结果、复现报告。

## 安装教程

### 如果直接用我下载好的数据和模型来复现：

RGCL-main.tar.gz的解压目录——RGCL/RGCL-main
RGCL_Models.tar.gz的解压目录——RGCL/RGCL_Models
RGCL_Full_Environment.tar.gz的解压目录——RGCL/offline_packages

项目主代码的位置——RGCL\RGCL-main\RGCL-main\src
训练和测试数据的位置——RGCL\RGCL-main\RGCL-main\data
测试文件的位置——RGCL\RGCL-main\RGCL-main\scripts
测试结果的位置——RGCL\RGCL-main\RGCL-main\results

### 如果需要复现下载数据和模型的过程，在可以联网的WSL端运行download_data.py和download_models.py，其它的同上。
